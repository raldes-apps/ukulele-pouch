FAQ

Why do I get multiple chord names for one shape?

Some fingerings can be interpreted in different ways (sus, add, inversions). The app lists several valid names that match the same set of notes.

What if the shape returns no result?

Double-check open strings and fret positions. If it is truly outside common chord structures, it may not map cleanly to a standard chord name.

Is it beginner-friendly?

Yes. Even without theory, it helps you name what you play and learn faster.
